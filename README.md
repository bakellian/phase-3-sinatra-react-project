# The to-do list Backend
This is the backend for a to-do applicaiton where users can add, edit, and delete to-do's in their list. This app uses ruby and Active Record. 

### Requirements
ruby 2.7.4
sqlite3 3.37.0


### How to use 
Fork and clone. 
bundle install
run rake db:migrate
run rake db:seed
run rake server
